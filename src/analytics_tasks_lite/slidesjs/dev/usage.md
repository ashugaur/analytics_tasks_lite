# Usage

## check_chart.py

python check_chart.py my_chart.html          # single file
python check_chart.py chart1.html chart2.html  # multiple
python check_chart.py charts/               # whole folder


## check_project.py
python check_project.py path/to/slidejs_control.xlsm
python check_project.py path/to/slidejs_control.xlsm --test-id test_1
python check_project.py path/to/slidejs_control.xlsm --skip-dry-run   # fast structural check only
